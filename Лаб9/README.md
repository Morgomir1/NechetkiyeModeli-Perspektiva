# Лабораторная работа: Система Такаги-Сугено и ANFIS

## Описание

Лабораторная работа по построению нечеткой системы Такаги-Сугено и адаптивной нейро-нечеткой системы вывода (ANFIS) для оценки кредитоспособности клиентов на основе таблицы Credit.

## Структура проекта

- `sugeno_credit_system.py` - построение системы Такаги-Сугено для данных Credit
- `anfis_training.py` - обучение ANFIS для настройки параметров модели
- `generate_plots.py` - генерация графиков и визуализаций
- `generate_lab_report.py` - генерация HTML отчета
- `run_all.py` - главный скрипт для запуска всех компонентов
- `run_lab.bat` - bat-файл для запуска на Windows

## Требования

- Python 3.7+
- numpy
- pandas
- matplotlib
- scikit-learn

Установка зависимостей:
```bash
pip install numpy pandas matplotlib scikit-learn
```

## Запуск

### Вариант 1: Запуск через bat-файл (Windows)
Двойной клик на `run_lab.bat`

### Вариант 2: Запуск через Python
```bash
python run_all.py
```

### Вариант 3: Пошаговый запуск
```bash
# 1. Построение системы Сугено
python sugeno_credit_system.py

# 2. Обучение ANFIS
python anfis_training.py

# 3. Генерация графиков
python generate_plots.py

# 4. Генерация HTML отчета
python generate_lab_report.py
```
