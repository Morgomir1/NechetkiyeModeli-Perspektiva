# Лабораторная работа по анализу временных рядов

## Описание
Лабораторная работа по анализу временных рядов с использованием моделей ARIMA.

## Исполнитель
Тимошинов Егор Борисович, группа 16

## Запуск
Для генерации HTML отчета запустите:
```
python generate_lab_report.py
```

Или используйте bat-файл:
```
run_lab.bat
```

## Результаты
После выполнения скрипта будет создан файл `Timoshinov_E_B_Lab_Timeseries.html` с результатами анализа.

## Требования
- Python 3.7+
- numpy
- pandas
- matplotlib
- statsmodels
- scipy

Установка зависимостей:
```
pip install numpy pandas matplotlib statsmodels scipy
```
