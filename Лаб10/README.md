# Лабораторная работа 10: Сравнение методов Мамдани и Сугено

## Описание

Лабораторная работа по сравнению двух основных методов нечеткого вывода: Мамдани и Сугено. Анализ особенностей каждого метода, их преимуществ и недостатков на примере задачи управления с двумя входными переменными.

## Структура проекта

- `fuzzy_comparison_lab.py` - основной скрипт с реализацией систем Мамдани и Сугено
- `generate_plots.py` - скрипт для генерации графиков
- `generate_lab_report.py` - скрипт для генерации HTML отчета
- `run_all.py` - главный скрипт для запуска всех этапов последовательно
- `run_lab.bat` - bat-файл для запуска на Windows

## Требования

- Python 3.6+
- numpy
- matplotlib

Установка зависимостей:
```bash
pip install numpy matplotlib
```

## Описание системы

Система имеет две входные переменные:
- **Входная переменная 1**: от 0 до 10
- **Входная переменная 2**: от 0 до 10

Выходная переменная:
- **Выходное значение**: от 0 до 20

Для каждой входной переменной определены три терма: низкое, среднее, высокое.

Система содержит 9 правил вывода, охватывающих все комбинации входных термов.

3. **Сравнение методов**
   - Тестирование на различных входных значениях
   - Анализ разницы между методами
   - Визуализация результатов

4. **Анализ результатов**
   - Преимущества и недостатки каждого метода
   - Области применения
