# Лабораторная работа 10: Марковские цепи для студентов

## Описание
Определение матрицы переходных вероятностей и решение векторно-матричного уравнения (10)-(11) для нахождения стационарного распределения для двух студентов.

## Уравнения
- **Уравнение (10)**: π = πP
- **Уравнение (11)**: Σπᵢ = 1

где π - вектор стационарных вероятностей, P - матрица переходных вероятностей.

## Запуск

### Вариант 1: Использование bat-файла
Запустите `run_lab.bat` для выполнения вычислений или `run_all.bat` для полного выполнения (вычисления + генерация HTML).

### Вариант 2: Прямой запуск Python скриптов
1. Выполните вычисления:
   ```bash
   python lab10_calculation.py
   ```

2. Сгенерируйте HTML отчет:
   ```bash
   python generate_lab_report.py
   ```

3. Или запустите все сразу:
   ```bash
   python run_all.py
   ```

## Изменение данных

Если данные в PDF отличаются от тех, что в скрипте, откройте файл `lab10_calculation.py` и измените словари `student1_transitions` и `student2_transitions`.

Формат данных:
```python
student1_transitions = {
    (i, j): count,  # количество переходов из состояния i в состояние j
    ...
}
```
