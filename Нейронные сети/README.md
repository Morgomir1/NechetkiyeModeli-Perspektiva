# Лабораторная работа: Нейронные сети

## Описание

Лабораторная работа по построению и обучению нейронных сетей для прогнозирования прочности бетона.

## Требования

Для выполнения лабораторной работы необходимо установить следующие библиотеки:

```
pip install numpy pandas matplotlib scikit-learn
```

Примечание: Используется sklearn MLPRegressor, который не требует установки TensorFlow.

## Запуск

Для запуска лабораторной работы выполните один из следующих способов:

1. Запуск через bat-файл (Windows):
   ```
   run_lab.bat
   ```

2. Запуск через Python:
   ```
   python run_lab.py
   ```

3. Пошаговый запуск:
   - Сначала выполните основной скрипт:
     ```
     python neural_network_lab.py
     ```
   - Затем сгенерируйте HTML-отчет:
     ```
     python generate_lab_report.py
     ```

## Результаты

После выполнения лабораторной работы будут созданы:

- `Timoshinov_E_B_Lab_NeuralNetworks.html` - HTML-отчет с результатами
- `predictions_model1.png`, `predictions_model2.png`, `predictions_model3.png` - графики предсказаний
- `predictions_comparison.csv` - таблица сравнения результатов
- `neural_network_results.pkl` - сохраненные результаты для отчета

## Выполненные задачи

1. Загрузка и нормализация данных о прочности бетона
2. Построение нейронной сети с 1 скрытым узлом
3. Построение нейронной сети с 3 скрытыми узлами
4. Построение нейронной сети с кастомной функцией активации smReLu
5. Оценка качества моделей с помощью корреляции
6. Визуализация результатов
7. Генерация HTML-отчета
