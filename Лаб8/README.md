# Лабораторная работа № 8: Проектирование нечетких систем Суджено

## Описание

Данная лабораторная работа реализует нечеткую систему вывода Суджено для управления температурой.

## Структура проекта

- `sugeno_fuzzy_system.py` - основной скрипт с реализацией системы Суджено
- `generate_plots.py` - скрипт для генерации графиков
- `generate_lab_report.py` - скрипт для генерации HTML отчета
- `run_all.py` - скрипт для запуска всех этапов последовательно

## Требования

- Python 3.6+
- numpy
- matplotlib

Установка зависимостей:
```bash
pip install numpy matplotlib
```

## Запуск

### Вариант 1: Запуск всех этапов последовательно
```bash
python run_all.py
```

### Вариант 2: Запуск по отдельности

1. Создание и тестирование системы:
```bash
python sugeno_fuzzy_system.py
```

2. Генерация графиков:
```bash
python generate_plots.py
```

3. Генерация HTML отчета:
```bash
python generate_lab_report.py
```

## Выходные файлы

После выполнения всех скриптов будут созданы следующие файлы:

- `sugeno_results.pkl` - результаты работы системы (pickle файл)
- `temperature_membership.png` - график функций принадлежности для температуры
- `dT_membership.png` - график функций принадлежности для скорости изменения температуры
- `sugeno_surface.png` - 3D поверхность вывода системы
- `test_results.png` - график результатов тестирования
- `Timoshinov_E_B_Lab_Sugeno.html` - финальный HTML отчет

## Описание системы

Система имеет две входные переменные:
- **Температура (T)**: от 0 до 50°C
- **Скорость изменения температуры (dT)**: от -5 до +5 град/мин

Выходная переменная:
- **Мощность нагревателя (P)**: от 0 до 100%

Система содержит 9 правил вывода Суджено с константными выходными значениями.
